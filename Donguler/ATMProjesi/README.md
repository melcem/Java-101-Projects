# ATM Projesi
* Java döngüler ile kullanıcının banka hesabını yönetebildiği bir ATM projesi yapıyoruz.


## Ödev
* Aynı projedeki ATM işlemlerini "Switch-Case" kullanarak yapınız.
***
## Senaryo 1
  - Bakiye = 1500;
  - Yapilacak islem: 1;  
  - Cekilecek miktar: 1000;
  - Yapilacak islem: 3;
  - Bakiye: 500;
  
## Senaryo 2
  - Bakiye = 1500;
  - Yapilacak islem: 2;  
  - Cekilecek miktar: 2500;
  - Yapilacak islem: 3;
  - Bakiye: 4000;
 
## Senaryo 3
  - Bakiye = 1500;
  - Yapilacak islem: 4;
  - Iyi gunler...
